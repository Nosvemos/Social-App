## Environment Variables

To run this project you will need to add the following environment variables to your .env file.

```
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
DATABASE_URL=

NODE_ENV=production (optional)
```

## Run the app

```shell
npm run dev
```
